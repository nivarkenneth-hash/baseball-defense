VERSIÓN ACADEMIA DEFENSIVA

Incluye:
- 200 jugadas defensivas.
- Modo interactivo “¿Dónde tiro?”.
- Situaciones de fuerza y doble play.
- Animación de la secuencia de tiros.
- Puntuación de aciertos.

PARA ACTUALIZAR GITHUB:
1. Extrae este ZIP.
2. En tu repositorio baseball-defense abre Add file → Upload files.
3. Sube y reemplaza todos estos archivos:
   index.html
   plays.js
   situations.js
   service-worker.js
   manifest.webmanifest
   icon-192.png
   icon-512.png
4. Pulsa Commit changes.
5. Espera 1–3 minutos.
6. Abre el mismo enlace de GitHub Pages.
7. Si ves la versión anterior, actualiza la página o elimina el acceso anterior y vuelve a agregarlo.
